400

Bad Request

Błąd 400 oznacza, że żądanie wysłane przez ciebie do serwera jest niepoprawne. Mówiąc prościej, otrzymaliśmy niekompletne lub puste żądanie, które niezawiera danych wymaganych do wykonania żądania.

Przyczyny błędu 400:

  • Próba zamodzielnej zabawy API.
  • Próba obejścia zabezpieczeń API.
  • Nieznajomość struktury zapytań.
  • Błąd w kodzie JavaScript strony.

Rozwiązanie problemu 400:

  • Jeśli żądanie, które spowodowało ten błąd zostało wysłane przez ciebie własnoręcznie za pomocą programu takiego jak Insomnia, zaprzestań wysyłania żądań do naszsego serwera mając na uwadze fakt, że blokujemy wszystkie adresy IP, które nadużywają naszego API.
  • Jeśli po prostu korzystasz z naszej strony, poczekaj aż błąd zostanie naprawiony.

Wskazówki:

  • Nie próbuj samodzielnie naprawić błędu, gdyż możesz w taki sposób spowodować, że twój adres IP zostanie zablokowany.
